# QuantityMeasurementApp

A Java application demonstrating quantity measurement with unit conversions.

## Features
- Length measurement (Feet, Inches, Yards, Centimeters)
- Weight measurement (Kilogram, Gram, Pound)
- Volume measurement (Litre, Millilitre, Gallon)
- Temperature measurement (Celsius, Fahrenheit, Kelvin)
- Spring Boot REST API
- JWT & OAuth2 Authentication
- Microservices Architecture
